# < h1~6 > < /h1~6 >
hタグはheading(見出し)の意味である。表示結果からもわかるようにhタグは< /h>で閉じなくともhタグが先頭にある行は一つの見出しになる。
構造と表記を意味するタグ。

記述例 [](変更しない)

```
<body>
  <h1>H1_H1</h1>
  <h2>H2_H2
  <h3>H3_H3
  <h4>H4_H4
  <h5>H5_H5</h5>
  <h6>H6_H6</h6>
</body>
```
