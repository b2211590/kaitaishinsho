# < BODY > < /BODY >
構造を意味するタグ、文書の本文であることを表す。テキスト、画像、リンク、動画などを明記的にメインコンテンツのために使用されるタグである。bodyタグ内に書かれたコンテンツは、実際にブラウザ上で表示される。

記述例 [](変更しない)
  
  ```
    <BODY>
    <p>Hello123</p>
    <B> Hello </B>
    </BODY>
  ```
